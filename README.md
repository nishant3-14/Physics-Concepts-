# Physics Concepts

This repository is a collection of self-contained notes aimed at developing a conceptual and geometric understanding of core ideas in physics.

Rather than focusing only on formal derivations, these notes emphasize:

* Geometric intuition
* Physical interpretation
* Structural understanding of mathematical objects

## Topics Covered

* Vectors, Covectors, and Tensors
* Lorentz Transformations
* Minkowski Geometry
* (More to be added)

## Motivation

While studying advanced physics, I found that many concepts are often learned computationally but not deeply understood. This repository is an attempt to reconstruct these ideas from first principles, focusing on meaning rather than memorization.

Each section is written as a standalone document, but together they build toward a coherent understanding of spacetime and modern physics.

## Structure

Each topic lives in its own folder under `concepts/`, containing:

* A LaTeX source file
* A compiled PDF
* Supporting figures (if needed)
